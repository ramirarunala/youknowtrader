🚀 Opsi 1: GitHub Pages (Paling Recommended)

Step 1: Buat Akun GitHub

  1. Buka github.com
  2. Sign up pakai email
  3. Verifikasi email

Step 2: Buat Repository Baru

  1. Klik tombol "+" di pojok kanan atas → "New repository"
  2. Isi Repository name: youknowtrader (atau apa aja)
  3. Pilih Public (biar gratis)
  4. Centang "Add a README file" (optional)
  5. Klik "Create repository"

Step 3: Upload File

  1. Di repository, klik "Add file" → "Upload files"
  2. Drag semua file HTML (9 file tadi) ke kotak upload
  3. Klik "Commit changes"

Step 4: Aktifkan GitHub Pages

  1. Di repository, klik "Settings" (tab di atas)
  2. Di sidebar kiri, klik "Pages"
  3. Di "Source", pilih "Deploy from a branch"
  4. Pilih "main" branch, folder "/ (root)"
  5. Klik "Save"

  1. Buka netlify.com
  2. Sign up pakai email/GitHub

  1. Klik "Site settings"
